How this works:

This is a Gradient Boosting Machine. It's a learning model which works on making a first prediction, and then working to minimize the residual errors when being trained - boosting the 'gradient' or differences created by each feature's values. It is best suited for instances where decisions are to be made, and the output is boolean. This particular one accepts csv (comma separated values) data to train on, and makes a tree. And when asked to predict,(again, in the form of csv data), generates predictions to the boolean value. The expected training data may look like:

Feature1,Feature2,Truth
25,18,0
25,22,0
24,23,0
23,35,1
28,38,0
30,39,1
31,29,1
36,40,1

Note that the truth value (the boolean value which is your expected output) must be entered as the last column in the csv data. Also note that the feature data accepted must be numerical. (Entries such as "low" "medium" and "high" can be split into numbers like 1,2,3 before entering.) The expected output (during training) must be either 0 or 1.
